# Zineland

## 简介

Zineland 是一个简单有效的杂志构建框架，能够快速通过 Markdown 文档生成杂志风格的静态网页

## 获取

- 源代码仓库 <https://github.com/zineland/zine.git>
- 获取
  - cargo

  ``` sh
    cargo install zine
  ```

  - brew

  ``` sh
    brew install zineland/tap/zine
  ```

- 文档 <https://zineland.github.io>

## 优点

- 移动端阅读体验优先
- 优雅的杂志设计
- 可定制化程度高，支持扩展
- 支持I18n和I10n
